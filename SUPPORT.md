# Support

## Questions and usage help

Use GitHub Discussions for usage questions, implementation ideas, and general support when Discussions are enabled for the repository.

If Discussions are not enabled, open a GitHub issue only when you can describe a concrete bug or actionable improvement.

## Bug reports

Please include:

- WordPress version
- PHP version
- Polylang version
- Active theme
- Steps to reproduce
- Expected behavior
- Actual behavior

## Security issues

Do not report security issues publicly. Follow the process in [SECURITY.md](SECURITY.md).
