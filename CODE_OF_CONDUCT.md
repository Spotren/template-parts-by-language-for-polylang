# Code of Conduct

## Our standard

Participation in this project should be respectful, constructive, and focused on improving the plugin for everyone.

Examples of expected behavior:

- Being respectful in discussions and reviews
- Giving clear, technical, good-faith feedback
- Accepting that contributors may have different levels of experience
- Focusing criticism on ideas, code, and behavior, not people

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or inflammatory comments
- Repeatedly derailing technical discussions
- Publishing private information without permission

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and review comments.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that do not follow this code of conduct.

## Reporting

For conduct concerns, contact the maintainer privately through the support or security contact path documented in this repository.
